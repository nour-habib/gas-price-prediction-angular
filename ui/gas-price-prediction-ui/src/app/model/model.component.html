Polynomial Regression Model

The polynomial regression model used in this experiment has the following form,

y = β 0 + β 1x + β 2x^2 + ... + β n x n + ∊ ,

which is linear in the unknown parameters βi. This model is a type of regression that can be viewed in a least squares sense where the sum of the squared residuals is minimized.


The Matlab function used to create the model is polyfit which accepts an n x p array of indendepent variables, (where n is the number of data points and p is the dimension of the variable space), dependent variablees as a n x 1 column or row vector and the model terms in the form of a string containing specific terms or, in our case, an integer k designating the highest degree in the objective function. I tested in models of degree k=1, 2, 3, 4, but the only the results for k=2, 3 are shown.
